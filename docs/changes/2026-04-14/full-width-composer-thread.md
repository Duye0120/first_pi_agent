# 调整聊天框至全宽适配

**日期:** 2026-04-14
**时间:** 16:45:00

## 改了什么
去除了聊天区和底部输入框（Composer）原有的宽度限制：
- 修改了 `thread.tsx` 中的全局 CSS 变量 `--thread-max-width`，将其值从 `56rem` (即 896px) 更改为了 `100%`。

## 为什么改
用户在聊天过程中觉得两侧留白太多并提到“让底部的聊天框撑满整个聊天区域”。通过取消默认的最大宽度约束并改为 100%，使得消息区与底部输入框都能自适应地横向撑满整个右侧聊天面板，以提供更宽广的长文及代码阅读体验。

## 改到哪些文件
- `src/renderer/src/components/assistant-ui/thread.tsx`
