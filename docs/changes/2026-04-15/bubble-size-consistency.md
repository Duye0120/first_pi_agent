# 聊天气泡内边距与大小差异优化

**时间**：2026-04-15 10:30:00

## 更改内容

- **修复了 User 与 Assistant 对话行在视觉上体量感失衡与边缘偏移的问题**：
  - 减小了用户提问气泡的空隙结构：将原本过于宽大的 `py-3.5 px-4` 紧凑化为 `py-2 px-4`，缩小其气泡的灰色框占地感。
  - 将 AI (`AssistantMessage`) 的文本内边距从原先极小缩减的 `px-1 py-1` 放大为 `px-2 py-2`，确保机器输出与用户回复不至于因为 Padding 过小显得行高拉跨。
  - 缩减了两方的整体视觉差异，解决了“用户气泡过于巨大”、“行高和占地与机器文本视觉不一致”的问题。
  - 除去了用户侧气泡外部容器上多余的 `px-2` 限制以保证它们各自能紧贴 Viewport 左侧（距两端）的统一 `px-6` 逻辑边界边缘对齐。
