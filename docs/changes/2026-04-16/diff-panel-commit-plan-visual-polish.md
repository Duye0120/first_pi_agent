# Diff Panel 提交计划视觉收口

**时间**: 2026-04-16 19:18

## 改了什么

1. 把“未生成计划”区域从大块占位卡改成轻提示条。
2. 重做了提交计划卡片的分区：标题、说明、原因、文件列表、操作区层级更清楚。
3. 标题输入区改成更稳定的双行高度，去掉小滚动条感。
4. 文件路径按钮改成更克制的面板标签样式，超长路径走截断展示。

## 为什么改

- 用户明确指出空提示块太大，体感像可输入区域。
- 旧卡片的表单区、路径区和按钮区挤在一起，界面观感粗糙。
- 标题区域出现滚动条会直接拉低质感。

## 涉及文件

- `src/renderer/src/components/assistant-ui/diff-panel.tsx`

## 结果

- 空状态更轻，阅读负担更低。
- 提交卡片的结构和质感更接近全局控制面板。
