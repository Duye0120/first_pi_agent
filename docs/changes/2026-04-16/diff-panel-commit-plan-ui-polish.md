# Diff Panel 提交计划 UI 收口

**时间**: 2026-04-16 19:02

## 改了什么

1. 去掉了提交计划面板里的批量 `全部暂存` 和 `依次提交全部` 按钮。
2. 保留逐组 `暂存本组` 和 `提交本组`，减少和左侧文件区顶部操作的重复感。
3. 把提交计划容器、卡片和输入区的圆角、阴影、焦点态往全局控制面板样式对齐。
4. 给计划卡片里的标题、说明和文件跳转按钮补了可访问性标签。

## 为什么改

- 底部批量动作和顶部文件区动作语义接近，界面里同时出现会让层级变乱。
- 提交计划区更适合强调“逐组确认和逐组提交”，避免一次给太多主操作。
- 用户要求这块 UI 和全局风格更一致，圆角和输入区质感需要继续收口。

## 涉及文件

- `src/renderer/src/components/assistant-ui/diff-panel.tsx`

## 结果

- 提交计划区的操作密度更低，主次更清楚。
- 卡片和输入区的圆角、焦点反馈更接近全局样式。
