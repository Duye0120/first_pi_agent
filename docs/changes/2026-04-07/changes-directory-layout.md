## 目录规则

- 目录格式：`docs/changes/YYYY-MM-DD/`
- 文件格式：`xxx.md`
- 时间写法：写在 md 正文顶部，不写进文件名

- 文件名不带分钟秒数，可读性更高
- 规则固定后，后面不容易再把 changes 写乱
